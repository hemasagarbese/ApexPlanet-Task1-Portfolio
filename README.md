# 🌐 Personal Portfolio Website

A modern, responsive, and interactive **Personal Portfolio Website** developed as part of **Task 1** of the **Full Stack Web Development Internship** at **ApexPlanet Software Pvt. Ltd.**

This project showcases my profile, technical skills, education, projects, internships, and contact information through a clean, responsive, and user-friendly interface built using HTML5, CSS3, and JavaScript.
